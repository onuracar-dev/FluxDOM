Viewport: desktop (1280x800)
Description: Verifies the product message and interactive model render successfully.

## Test 2 - Empty playground input is guarded
Viewport: desktop (1280x800)
Description: Verifies blank items are not added to the preview.
Steps:
1. Navigate to the playground

3. Click "Add"
Expected: The list remains unchanged and no blank list row appears.
